## Project

**controle de contas** — aplicativo de controle de contas pessoais (finanças pessoais). O objetivo é permitir que o usuário acompanhe receitas, despesas e saldo de suas contas do dia a dia.

## Stack

- **Framework:** Next.js (App Router)
- **Estilização:** Tailwind CSS

Sempre use Next.js e Tailwind CSS neste projeto, independentemente da funcionalidade sendo implementada.
